# WorkSolo V1.0.1 发布说明

## 简介

WorkSolo 是一个面向个人的 Windows 轻量工作事项工具，用于记录事项、推进处理、标记完成状态，并按时间和项目维度进行复盘总结。

## 当前版本包含

- 工作台概览
- 事项新增、编辑、删除
- 项目新增、编辑、删除
- 状态流转与异常标记
- 到期与逾期提醒
- 日 / 周 / 月总结复盘
- 中文界面与基础设置
- 固定数据目录与自动迁移
- 检查更新 / 版本历史页面
- 版本号正式固化为 `v1.0.1`

## 数据兼容

- 新版数据固定保存在 `AppData\Local\WorkSolo\data.json`
- 如果旧版数据在程序目录下的 `data\data.json`，首次启动会自动迁移
- 保存前会自动保留最近几份备份

## 使用方式

1. 解压压缩包
2. 双击根目录下的 `WorkSolo.exe`
3. 程序会自动启动 `AppLive` 中的主程序

## 说明

- 当前版本为本地单机版
- 不包含多人协作、云同步和账号体系
