# WorkSolo

![WorkSolo Logo](Assets/worksolo-mark.png)

WorkSolo 是一个面向个人使用的 Windows 轻量桌面应用，用来记录工作事项、推进处理过程、标记完成状态，并按时间和项目维度做闭环复盘。

当前仓库为 `V1` 首个正式版本源码。

## 产品定位

WorkSolo 聚焦一件事：

- 让个人工作事项从“记下来”走到“处理完”
- 让每天、每周、每月都能快速复盘
- 保持本地保存、轻量启动、低学习成本

## 当前功能

- 工作台：查看今天到期、已逾期、最近待推进事项
- 事项清单：支持新增、编辑、删除、状态流转、搜索和筛选
- 项目分类：支持新增、编辑、删除和启用停用
- 总结复盘：按日、周、月统计事项推进情况
- 设置页：主题、默认启动页、提醒开关、版本信息
- 本地数据：默认保存在项目根目录下的 `data/data.json`

## 技术栈

- `.NET 8`
- `WinUI 3`
- `Windows App SDK`
- 本地 `JSON` 持久化

## 目录说明

- `Pages/`：页面视图
- `ViewModels/`：页面状态与交互逻辑
- `Models/`：数据模型
- `Services/`：状态管理、存储、统计逻辑
- `Styles/`：主题和样式资源
- `Assets/`：图标与界面资源
- `Tools/`：辅助启动器源码
- `使用说明.md`：项目内中文使用手册

## 本地运行

### 方式一

直接双击项目根目录下的：

```text
WorkSolo.exe
```

### 方式二

双击：

```text
Run-WorkSolo.cmd
```

### 方式三

如果你在开发环境中自行构建：

```powershell
dotnet build .\WorkClosure.csproj -p:Platform=x64
```

## 数据说明

- 运行数据默认位于：`data/data.json`
- 仓库默认不提交本地运行数据
- 发布和开发使用的是同一套本地数据结构
